# reko Decompiler

![reko_logo](https://github.com/uxmal/reko/raw/master/web/images/reko.png)

リポジトリ: <https://github.com/uxmal/reko>

rekoは[John Källén氏](https://github.com/uxmal)によってC#で実装されているオープンソースのデコンパイラです.

詳細は把握していませんが, 10年近く個人開発を継続しているようで, いろいろな機能があるようです.

[ドキュメント用のWebサイト](https://uxmal.github.io/reko/)も用意されているようですが, メンテナンスされていない雰囲気があります.

## 余談

個人開発ということで, 実装を参考にしようかと思いましたが, C#で書かれていることやプロジェクトが巨大すぎることから名前だけ把握しておくだけにとどめています.
