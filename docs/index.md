## デコンパイラとは

デコンパイラとは, **De-Compiler**, つまり字義どおりに受け取ればコンパイルの逆を行うソフトウェアのことです. デコンパイラという言葉は非常に曖昧なもので, 研究者によっても明確な定義は異なります. 
広義的にこの言葉を定義すれば, **低レベルな機械語に近い言語を, 高レベルの表現に変換するソフトウェア**ということができます. 
低レベルな機械語に近い言語とは, 例えば単に機械語, アセンブリ言語, 仮想マシンのバイトコード等が挙げられます. 一方で高レベルの表現というのは, 高級なプログラミング言語(C言語等), 中間表現(例えばP-codeのような), 論理式, 自然言語によるロジックの説明等が挙げられます. 

デコンパイラの使途としては以下のようなものが挙げられます. 

- ソフトウェアリバースエンジニアリング (SRE)
- マルウェア解析
- 脆弱性調査
- ソフトウェアのリホスティング
- etc...

とにかく, 実行ファイルとしては入手できるが, ソースコードが入手できないソフトウェアの解析に用いられることが主です.
