# ICFG

ICFGはInter-Procedural Control Flow Graphの略語で[^1][^2], [CFG](./CFG.md)にプログラムに登場する関数間の呼び出し関係をエッジとして追加したものです[^3]. 

[^1]: `IPCFG`と呼ぶ派閥もあるようです. 
[^2]: なんならIntra-Procedural Control Flow Graphという用語もあるみたいです. 参考: https://www.usenix.org/legacy/publications/library/proceedings/sec04/tech/full_papers/kruegel/kruegel_html/node6.html
[^3]:2025年12時点ではこの定義を信じていますが, ICFGはあまりメジャーな概念ではない様で, 調べてもやや定義がふわふわしている用語に見えます.
